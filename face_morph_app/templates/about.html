About This Project

This project focuses on Face Morphing Attack Detection using machine learning techniques. Face morphing attacks involve blending two facial images to create a hybrid image that can impersonate both individuals, posing a serious threat to biometric security systems.

We have developed a web-based system that allows users to upload images or capture them using a live webcam. These images are processed and analyzed in real-time using a trained machine learning model. The model detects whether the input image is a genuine (real) image or a morphed (fake) one.

Technologies used in this project include Python, Flask, OpenCV, HTML/CSS/JS, and scikit-learn. The image is preprocessed and fed into a trained model that has been optimized with feature scaling and dimensionality reduction techniques to ensure high accuracy.
